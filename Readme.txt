Hotel Review Project

Welcome to the Hotel Review Project! This repository houses a comprehensive web application for managing hotel and campground reviews. Users can create accounts, leave reviews for various establishments, and upload images. The application also features user authentication using Passport.js.

Key Features:
- User Management: Create and manage user accounts to access the platform.
- Review System: Leave reviews for hotels and campgrounds, providing valuable feedback.
- Image Management: Upload, update, and delete images associated with hotels and campgrounds.
- User-Specific Deletion: Users can delete their own reviews for added control and privacy.
- Passport Authentication: Secure user authentication using Passport.js for a protected user experience.

This project utilizes MongoDB as the database and includes an intuitive user interface for easy navigation and interaction. It is built with modern web technologies, including JavaScript, Node.js, Express.js, and MongoDB.

Contributions, bug reports, and feature requests are always welcome. Feel free to explore the code and contribute to the ongoing development of the project. Start reviewing hotels and campgrounds today!
